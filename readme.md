Forest Adventure Game

A console-based text adventure game demonstrating data structures and algorithms.

Data Structures Used:
- Dictionary: Player data, items database, game flags
- List: Inventory management

Algorithms Used:
- Linear Search: Finding items in inventory
- Random Number Generation: Combat damage, sneak chances
- Conditional Logic: Game state management

## How to Run

### Prerequisites
- Python 3.6 or higher

### Running the Game

1. Open a terminal/command prompt
2. Navigate to the project folder:
   ```
   cd dsafinalproject
   ```
3. Run the game:
   ```
   python adventure.py
   ```

### Gameplay
- Follow the on-screen prompts to navigate through the forest
- Make choices by entering the corresponding number
- Goal: Defeat the witch and escape the forest!
